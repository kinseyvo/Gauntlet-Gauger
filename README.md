# Gauntlet Gauger
The Gauntlet Gauger is a wearable distance detector measurement utility. It detects the distance between the user and something and displays it on the LCD screen. The concept is to have a glove with the sensor on top of it, preferably on the wrist area. It is inspired by Batmanâ€™s gauntlet in media such as films, books, and video games.

## Software Used
- Arduino IDE

## Hardware Used
- UNO R3 Controller Board
- HC-SR04 Ultrasonic Sensor
- LCD1602 Module (with pin reader)
- Breadboard
- Breadboard Jumper Wires
- USB Cable
- Battery
- Battery Connector Cord
